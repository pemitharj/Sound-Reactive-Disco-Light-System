# Sound Reactive Disco Light System

An analog electronics project that converts real-time audio signals into dynamic LED lighting effects.

---

## Abstract
The Sound Reactive Light System is an analog electronics project designed to produce dynamic lighting effects that respond directly to audio input. The system analyzes different frequency bands of background music and drives LEDs in real time to create interactive visual effects for events and entertainment environments.

---

## Features
- Real-time audio-responsive lighting
- Frequency band separation (Bass, Mid, Treble)
- Analog signal processing (no microcontroller)
- Envelope detection and amplification
- LED visualization based on sound intensity

---

## System Overview
The system captures audio signals, amplifies them, and separates them into three frequency bands using analog filters. Each band is processed independently to drive LED outputs based on signal intensity.

---

## Technologies Used
- Analog Electronics
- Active Filters (Low-pass, Band-pass, High-pass)
- Operational Amplifiers
- PCB Design
- LTspice Simulation
- SolidWorks (Enclosure Design)

---

## Results
The system successfully produces real-time LED responses synchronized with music across bass, mid, and treble frequency ranges.
